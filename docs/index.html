Arxiv PDF to Google Drive



Downloads are stored on your google drive as: 

  [root] / [category] / [year] / [title].pdf
using the root folder you specified above and values of the remaining fields obtained from an arxiv API query. Notes:
  • The root folder must already exist and be unique on your google drive.
  • Files larger than 40Mb will not be downloaded.
  • Article titles will be truncated at 250 characters for the file name.
